油焖大虾的做法

发表于 2022-01-08 | 分类于 cook | | 阅读次数: | 浏览 次
字数统计: 362 | 阅读时长 ≈ 1

前言

做法简单，营养丰富，可以调理身体，促进发育，还可以防治高血压和心肌梗塞，孕产妇吃了还可以催乳，儿童吃了对抵抗力和身体发育都有很好的帮助。

准备

大虾1斤3两（大概650g）

香葱一把
姜一块
啤酒一罐
料酒
盐
白胡椒粉
糖

食材处理

虾

用剪刀减去虾头，减去虾的脚，去掉内脏和虾线，给虾开背，多洗几遍，洗干净

香葱

葱白：切段
葱叶：切葱花

姜

一半切片，一半切丝

制作

腌制

  1. 找个盆，放虾
  2. 加入1勺料酒，加入半勺盐
  3. 姜片和葱段用手挤出水
  4. 充分搅拌后，腌制10分钟

准备料汁

  1. 碗里加入三勺料酒，3勺生抽， 一勺白糖，半勺白胡椒粉
  2. 将调料搅匀

起锅烧油

  1. 腌制过的虾，挑出腌制的姜和葱
  2. 煎制（操作时间大概3-5分钟）
    • 倒油，油热放入虾
    • 不用着急翻动，将虾平铺开，一面熟了翻面
    • 翻面后，要用铲子一直压制虾，压出虾油，非常关键
    • 一直到煎到壳肉分离, 虾油味道足
  3. 炒制
    • 加入葱段姜丝炒香
  4. 焖
    • 加入刚调好的料汁
    • 加入200ml左右的啤酒
    • 盖上锅盖，焖煮3分钟
    • 开过收汁
  5. 到时间出锅, 撒上葱花。酥酥脆脆非常好吃
end
